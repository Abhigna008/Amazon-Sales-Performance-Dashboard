# Amazon Sales Performance Dashboard | Power BI Project

## 🔍 Executive Summary
A data-driven visualization project using Power BI to analyze a complex Amazon sales dataset. The project goal was to synthesize logistical and transactional data into a unified, executive-ready dashboard that tracks business growth, identifies bottlenecks in fulfillment, and highlights high-performing product segments.

## 📌 Features & Insights

| Focus Area | Visuals | Key Insights |
| :--- | :--- | :--- |
| **Logistics** | Pie Chart (Shipment %), Bar Chart (Courier Status) | Provides a clear breakdown of fulfillment efficiency, helping to identify delayed or cancelled orders that affect customer satisfaction. |
| **Inventory** | Bar Charts (Top Category, Size Performance) | Identifies the highest-velocity products and sizes to optimize stock levels and prevent inventory stockouts. |
| **Revenue** | Area Chart (Revenue Trends), Map (Sales by City) | Tracks sales trajectory over time and visualizes geographic sales hotspots for localized marketing focus. |

## 🛠️ Technical Implementation

### Data Cleaning (Power Query):
* Implemented **Advanced Cleaning** steps using Trim, Clean, and Remove Duplicates to ensure data integrity.
* Utilized **Change Type with Locale** for robust handling of date formatting errors.

### Data Visualization:
* Designed a custom, modern UI/UX layout (similar to leading Admin Dashboards) using Figma/Canva backgrounds for a seamless application look.
* Focused on **Color Psychology** and **Whitespace** to reduce visual clutter and prioritize data delivery.
